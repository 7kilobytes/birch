# birch

A [WIP] IRC client written in pure bash.

<img src="https://i.imgur.com/AbeKUg4.jpg" width="400px">

## Usage

```sh
birch [-s host -u nick -p pass -P port -c chan -x cmd]
```

## Dependencies

- `bash 4+`
